# 🏦 Loan Approval Analysis

A data analysis project focused on exploring and cleaning a loan approval dataset to uncover key patterns and factors that influence loan decisions.

---

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** and **Data Cleaning** on a loan approval dataset. The goal is to understand the distribution of applicant profiles, identify missing or inconsistent data, and visualize trends related to loan approval outcomes.

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| Pandas & NumPy | Data manipulation and cleaning |
| Matplotlib & Seaborn | Static visualizations |
| Plotly | Interactive charts |
| Jupyter Notebook | Analysis environment |

---

## 📊 Key Steps

- **Data Cleaning** — Handled missing values, corrected data types, and removed inconsistencies
- **EDA** — Analyzed distributions of income, loan amount, credit history, and more
- **Visualizations** — Created charts to compare approval rates across gender, education, employment status, and property area
